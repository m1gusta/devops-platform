# DevOps Platform

## Description

A demonstration DevOps project showing the deployment of a production-like infrastructure using Docker, Nginx, FastAPI, PostgreSQL, Redis, Prometheus, Grafana, GitHub Actions and Ansible.

## Stack

- Docker
- Docker Compose
- FastAPI
- PostgreSQL
- Redis
- Nginx
- Prometheus
- Grafana
- GitHub Actions
- Ansible
